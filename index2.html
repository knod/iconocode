Iconocode

Hotbar


Instructions

Press Cmd+i in a word in the editor to give that word an icon

--- Then press tab and the arrow keys to set the purpose of the variable

--- Then use the cursor to select images (most icons currently don't get displayed in the search bar)

--- To search for an icon, type text into the search bar (will only activate for more than 2 letters)

Press Shift+Enter while in the editor to apply your code

Click on the output element on the right to activate its event listeners
